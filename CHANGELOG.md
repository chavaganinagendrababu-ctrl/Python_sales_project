# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-26

### Added
- Initial project release
- Sales Dashboard with 5 key visualization charts
- Animated and static dashboard exports
- Comprehensive data analysis capabilities
- Professional project structure
- MIT License
- Requirements management

### Features
- Monthly Revenue Trend visualization
- Top 10 Products by Revenue ranking
- Revenue Distribution by Category analysis
- Price vs Profit Analysis scatter plot
- Revenue by Region breakdown
- Automated chart generation and export
- Support for both animated (GIF) and static (PNG) outputs

### Technical
- Python 3.7+ compatibility
- Object-oriented code structure
- Pandas-based data processing
- Matplotlib-based visualizations
- Comprehensive error handling
